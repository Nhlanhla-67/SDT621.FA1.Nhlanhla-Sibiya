# SDT621 - Formative Assessment 1

This repository contains the C# practical solutions for Formative Assessment 1 (SDT621) by Nhlanhla Sibiya.

### ⚠️ Technical Note Regarding Git Commits
*Due to local Git configuration issues on my machine during development, commits in this repository may appear under two different GitHub accounts. Please note that both accounts belong to me, and all code submitted is entirely my own work.*

## Projects Included:
1. **Student Results (Console App):** Calculates student average marks and determines PASS/FAIL status. 
2. **Simple ATM (Console App):** Simulates a basic banking withdrawal transaction. 
3. **Favourite Languages (WinForms App):** A GUI application to add and remove programming languages with validation. 
4. **Emfuleni Municipality (Console App):** An object-oriented system to capture and process resident service requests based on urgency. 
5. **Digital Identity Processor (WinForms App):** A GUI application that validates South African ID numbers, calculates age, and generates a citizen profile.

## How to Run:
* Open the solution in Visual Studio / Visual Studio Code.
* If using Visual Studio, right-click on the specific project you want to test in the Solution Explorer and select "Set as Startup Project", then press F5.
